# my-create-skills

A collection of Claude Code skills for creating and evaluating skills.

## Skills

### write-skill
从零创建一个高质量 Skill 文件。当用户说"写个 skill"、"创建 skill"、"新建 skill"、"帮我写一个 skill"、"skill 脚手架"时触发。与 eval-skill 组成 Write→Eval 工作流。

### eval-skill
对一个 Skill 文件执行红队压力测试、四维专业评价、TDD 式 Patch 加固。当用户说"评测 skill"、"测试我的 skill"、"skill 评分"、"跑一下这个 skill"时触发。
